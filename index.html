O que são Tags?

Tags são usadas para que possa entender o que deve ser feito, para passar para o navegador o que deve ser mostrado a quem está vendo a página

Algumas mais comuns são:

  • h1: para um título principal (header 1), seguindo consequentemente com h2, h3, h4, h5 e h6 como subtitulos
  • p: para uma parágrafo
  • a: para um link
