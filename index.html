O que são Tags?

Tags são usadas para que possa entender o que deve ser feito, para passar para o navegador o que deve ser mostrado a quem está vendo a página

Algumas mais comuns são:

  • h1: para um título principal (header 1), seguindo consequentemente com h2, h3, h4, h5 e h6 como subtitulos
  • p: para uma parágrafo
  • a: para um link
  • img: adicionar uma imagem


Atributos das Tags

Atributos servem para alterar ou adicionar alguma funcionalidade a tag

Alguns atributos comuns:

  • id: para definir um id para essa tag, deixando a única
  • style: para constumizar o elemento dessa tag visualmente
  • class: para definir um grupo no qual essa tag com o elemento se encontrará


Listas ordenadas e não ordenadas

Há dois tipos de listas: não ordenadas e ordenadas;
Para fazer as ordenadas usamos o comando "ol"(ordered list) e para não ordenadas "ul"(unordered list). Para listarmos os items usamos a tag "li"(list item)
